# MAS Architecture Project

Framework khởi đầu cho các tác vụ phân tích kiến trúc hệ thống, tối ưu hiệu năng và điều phối tác tử AI.

## Cấu trúc chính
- data/: lưu dữ liệu thô và dữ liệu đã xử lý
- src/: mã nguồn cốt lõi cho ETL, đồ thị kiến trúc và tác tử AI
- notebooks/: notebook phân tích và thử nghiệm
- credentials/: khóa xác thực bảo mật
